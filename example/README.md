# Example for @wzjiscode/leafer-x-bwip

运行 demo：

1. 安装依赖（项目根目录）：
   npm install

2. 构建插件（若使用源码开发）：
   npm run build

3. 启动静态服务器（示例）:
   npx http-server example -c-1

4. 打开浏览器访问 http://localhost:8080

说明：
- 如果 example/main.js 找不到 ../dist/index.js，请修改导入到你本地构建输出的位置或先运行构建。
- demo 使用 leafer-ui 的 CDN 版，仅用于本地快速验证。
